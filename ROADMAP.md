# Roadmap

## Version 1.0

- [x] Native C++23 CLI foundation
- [x] Modular architecture
- [x] Configuration and structured logging
- [x] Hardware detection
- [x] Model catalog and model manager
- [x] Basic local agent tools
- [x] Test harness
- [ ] Connect a production local inference backend
- [ ] Publish official model manifest with URLs and SHA-256 checksums
- [ ] Add CI for Windows, Linux, and macOS
- [ ] Add packaged release artifacts

## Future Versions

- Native GUI built on the same core modules
- Richer local automation with policy controls
- Optional web search when explicitly requested
- Deeper operating system integration
- HEXTA OS integration
- Plugin APIs after the core is stable

Out of scope for Version 1.0: GUI, vision models, speech models, image generation, advanced multi-agent orchestration, and unconfirmed automation.
