# Changelog

All notable changes to JARVIS Lite are documented here.

## 1.0.0 - 2026-07-17

### Added

- Initial C++23 CMake project structure.
- Interactive CLI with help, version, clear, exit, and session history.
- Streaming inference interface with local runtime boundary.
- Built-in model catalog for Gemma, Qwen Coder, and DeepSeek Coder targets.
- Model manager for listing, downloading, verifying, selecting, and removing models.
- Optional libcurl-backed HTTP downloader with resume support.
- SHA-256 utility for model integrity checks.
- Cross-platform hardware detection.
- Configuration manager with a small TOML-like file format.
- Structured JSON logger.
- File-system agent tools and confirmed shell command execution.
- Build, installation, user, and developer documentation.
- Basic test executable covering utilities, configuration, file agent, and model recommendation.
